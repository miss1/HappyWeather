# HappyWeather

> 毕业设计 天气预报 安卓原生实现<br>
> 天气接口使用的是和风天气<br>
> 用gson解析json数据<br>
> 用butterknife进行数据绑定

> [H5版代码请戳这里](https://github.com/miss1/HappyWeatherH5)<br>
